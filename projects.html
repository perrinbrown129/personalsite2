bingeworthy | Github


bingeworthy is a simple app that uses the TV Maze API to let users find a new "bingeworthy" television show based off of their favorite genre. Built with HTML, CSS, JavaScript and jQuery

GFD | Github

GFD is a small directory of gluten-free canned alcoholic beverages that are safe for those with Celiac disease. The site directs users to more information about the beverages, where they can purchase them and allows them to submit new drinks. Built with HTML, CSS, EJS and Node.js

L.G.B.T.S. | Github

L.G.B.T.S. (Let's Go Buy Trendy Stuff or LGBT+ Shops) is a website that features small LGBTQ+ owned online shops that are not usually featured on popular e-commerce hubs unless it's Pride month. Built with Rails, React, HTML, CSS & Postgresql
